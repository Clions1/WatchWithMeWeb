# 🎥 Watch With Me

Real-time screen sharing and chat application. Share your screen with friends and chat instantly!

## ✨ Features

- 🖥️ **Screen Sharing**
  - High-quality video streaming (1080p, 60fps)
  - Audio sharing support
  - Fullscreen mode
  - Volume control

- 💬 **Chat**
  - Real-time messaging
  - Participant list
  - Online/offline status
  - Readable message history

- 🔒 **Security**
  - Password-protected rooms
  - Unique room IDs
  - Secure connection (WebRTC)

- 🎨 **Modern Interface**
  - Responsive design
  - Light/dark theme support
  - User-friendly interface
  - Modern animations

## 🚀 Getting Started

### Requirements

- Node.js (v14 or higher)
- Modern web browser (Chrome, Firefox, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Clions1/WatchWithMeWeb.git
cd WatchWithMeWeb
```

2. Install dependencies:
```bash
npm install
```

3. Start the application:
```bash
npm start
```

4. Open in your browser:
```
http://localhost:3000
```

## 💡 Usage

### As a Broadcaster

1. Select "Broadcaster" role on the home page
2. Enter your username
3. Set a room password
4. Click "Create Room" button
5. Click "Start Stream" and select the screen you want to share
6. Share the room ID with viewers

### As a Viewer

1. Select "Viewer" role on the home page
2. Enter your username
3. Enter room ID and password
4. Click "Join Room" button
